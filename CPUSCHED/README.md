#CPUSCHED
Primary Executeable File:  CPUSCHED/src/cpuScheduler.java
In order to activate and execute this file, first change directories to the CPUSCHED section through the command:  cd CPUSCHED.  Then, on the command line, type this: java src/cpuScheduler.java.  This should activate the file in which you will get a prompt that says 'Select A Scheduling Style You Want to Simulate:  (FIFO/SJF)'.  Note that when typing an input for this, the command line input must be in all caps. 

## FIFO
First In First Out (FIFO)
### Instructions
In order to activate this command, when the prompt above appears, type 'FIFO' to choose this scheduler.  


## SJF
Shortest Job First (SJF)
### Instructions
In order to activate this command, when the prompt above appears, type 'SJF' to choose this scheduler.
